¿Cómo funciona?

Esta aplicación te ayudará a aprender comandos de Linux de forma práctica:


  1. Cada nivel tiene varios comandos que deberás practicar.

  2. Completa todos los comandos de un nivel para desbloquear su FLAG.

  3. Usa la FLAG para desbloquear el siguiente nivel.

  4. ¡Completa los tres niveles para convertirte en un experto de Linux!

Puedes usar el comando help en cualquier momento para obtener ayuda.
